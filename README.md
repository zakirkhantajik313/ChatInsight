# ChatInsight

ChatInsight is a modern WhatsApp chat viewer and analytics tool.

Features:

* WhatsApp-style chat interface
* Identity selection
* Search with highlighted results
* Sender filtering
* Message statistics
* Most active days tracking
* Calendar-based date navigation
* Media detection
* Call and system message detection
* Light and dark themes

Upload an exported WhatsApp `.txt` file, select your identity, and explore your conversations in an interactive interface.
